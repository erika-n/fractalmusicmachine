Fractal Music Machine is a fractal music generator and music experimentation platform. 
  
It contains:

FractalMusicMachine:
	A user interface for creating fractals. 
SoundLabBasics:
	A library of useful sound tools, including oscillators, wav file import/export, 
	sound buffer, and envelope. 
FractalProject:
	The actual fractal making engine. 
	Contains a command line utility for making fractals, and also is the back end for FractalMusicMachine.
